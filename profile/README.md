# Across Report Renderer (ACR)

**Render once. Output everywhere.**

ACR is a printer-independent, cross-platform document rendering engine.

## 🚀 Features

- Cross-platform (Windows / Linux / macOS)
- CLI-based rendering engine
- Template specification (ACR spec)
- High portability and flexibility

## 📦 Projects

- acr-cli — CLI rendering engine
- acr-spec — Template specification
- rpx2json — Converter tool
