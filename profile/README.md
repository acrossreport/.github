# Across Report Renderer (ACR)

**Render once. Output everywhere.**

ACR is a printer-independent, cross-platform document rendering engine.

ACR guarantees pixel-perfect output across screen and print, without relying on printer drivers.

---

## 🚀 Features

- Cross-platform (Windows / Linux / macOS)
- Pixel-perfect layout (WYSIWYG)
- Printer-independent rendering (no driver dependency)
- CLI-based engine for automation
- Template specification (ACR spec)
- High portability and deterministic output

---

## 📦 Projects

- acr-cli — CLI rendering engine
- acr-spec — Template specification
- rpx2json — Converter tool

---

## 🔧 Architecture

Template → Layout Engine → Rendering Engine → Output

---

## 📄 Output Formats

- PDF
- PNG / Image
- Thermal printer commands (planned)
- Screen rendering
- Future device-specific renderers
